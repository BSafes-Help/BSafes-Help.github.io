# Create your first page

1. Select Personal.
2. Click blue + button.
3. Select Page.
4. Enter title for the page, click CREATE.
5. On a page, you could
  * add tags;
  * edit title and content;
  * create gallery of images;
  * attach files; and
  * comment on a page.
